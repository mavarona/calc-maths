# Maths Operators

Module to do basic maths operation

## Installation

follow the next instructions

```

npm install calc-maths
```

## Use

follow the next instructions

```

// Import module
const m = require('.');

// Sum
console.log(m.sum(2, 3)); => 5
// Substract
console.log(m.substract(2, 3)); => -1
// Multiply
console.log(m.multiply(2, 3)); => 6
// Divide
console.log(m.divide(27, 3)); => 9
```
